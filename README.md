Cotafei Portfolio 🎮

> Личный сайт-портфолио в стиле Undertale

![Avatar](https://avatars.githubusercontent.com/cotafei)

## ✨ Особенности

- 🎵 Звуки как в Undertale (при наведении, кликах)
- 📦 Автоматическая загрузка репозиториев с GitHub
- 🖥 Пиксельная графика и анимации
- 📱 Адаптивный дизайн
- ⚡ Интерактивные карточки проектов

## 🛠 Технологии

- HTML5
- CSS3 (Pixel-art стилизация)
- JavaScript (ES6+)
- GitHub API

## 🚀 Быстрый старт

1. Клонируй репозиторий:
```bash
git clone https://github.com/cotafei/cotafei.github.io
```

2. Открой `index.html` в браузере

Или просто перейди по ссылке: [cotafei.github.io](https://cotafei.github.io)

## 🎮 Фишки

- Звук при наведении на скиллы и кнопки
- Карточки репозиториев двигаются за мышкой
- Эффект старого монитора
- Анимация загрузки в стиле игр

## 📂 Структура

```
├── index.html          # Главная страница
├── style.css           # Все стили (Undertale тема)
├── script.js           # Логика и GitHub API
├── sounds/             # Папка со звуками
│   ├── snd_select.wav
│   ├── snd_tempbell.wav
│   └── snd_tempbell1.wav
└── README.md
```

## 🔧 Настройка

Если хочешь использовать для себя:
1. Замени `cotafei` в `script.js` на свой GitHub username
2. Поменяй ссылки на соцсети в `index.html`
3. Добавь свои звуки в папку `sounds/`
